Automatizovano Asset Bundlovanje

Razvio sam poseban Blender Addon za moj tim kako bi se standardizovale konvencije imena i automatizovao EXPORT/ORPHAN hijerarhija kolekcija.

  • Bez ručnog parentovanja: Skripta automatski detektuje tipove aseta i povezuje ih sa logičkim "Empties" na osnovu dinamičkih podataka grupe.
  • Automatsko povezivanje materiјala: Integrisan sa centralnom .blend bibliotekom da automatski dodeli produkcijski spreman materijal tokom faze preimenovanja.
  • Brzina radnog toka: Predstavljen prilagođeni Pie Menu sistem, smanjujući vreme imenovanja/organizovanja za približno 60%.
collection_bundler.py 

def bundle_selected_objects():
    selected_objs = [obj for obj in bpy.context.selected_objects if obj.type in {'MESH', 'CURVE'}]
    
    export_col = ensure_collection("EXPORT", color_tag='COLOR_04')
    placeholder = ensure_empty("PLACEHOLDER")

    for obj in selected_objs:
        name = obj.name.lower()
        
        for group, subs in GROUPS.items():
            if any(sub.lower() in name for sub in subs):
                # Automated Hierarchy Logic
                if group not in group_empties:
                    group_empties[group] = ensure_empty(group)
                    safe_link_to_collection(group_empties[group], export_col)
                    group_empties[group].parent = placeholder
                
                safe_link_to_collection(obj, export_col)
                obj.parent = group_empties[group]
